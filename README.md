
# Deel Automation Test

Candidate exam - Daniel Velazco


## Run Locally

### Pre-requisites
- Node v14.x.x

Clone the project

```bash
  git clone https://github.com/danielvelazco/deel.git
```

Go to the project directory

```bash
  cd deel
```

Install dependencies

```bash
  npm install
```

Run the test

```bash
  npm run wdio
```

Generate and display the report

```bash
  npm run generateReport
```
